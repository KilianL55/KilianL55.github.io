Présentation du projet :

Projet réalisé durant le stage de 1er année de BTS SIO consistant à réaliser le développement d'APIs pour l'entreprise Zaacom.

Développement des APIs WHOIS, Wayback, Majestic et Semrush

- L'API WHOIS sert à récupérer les informations d'un nom de domaine (date d’enregistrement, nom de la société, mail, adresse…)

- L'API Wayback sert à récupérer les captures d'écran d'un site web à une date voulue

- L'API Majestic sert à récupérer les informations d'un site web (nombre de backlinks, nombre de pages indexées, nombre de mots clés…)

- L'API Semrush sert à récupérer les informations d'un site web (nombre de backlinks, nombre de pages indexées, nombre de mots clés…)

Présentation du projet :
